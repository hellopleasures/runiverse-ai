# Get Battle Logs By UserID

GET 

## https://api-gateway.skymavis.com/origin/v2/community/users/:client\_id/battle-logs

Get paginated battle results for the specified user.

## Request[​](/api/origins/get-battle-logs-by-user-id#request "Direct link to Request")

### 

Path Parameters

**client\_id** stringrequired

Unique string that identifies a user.

**Example:** 1ec9eb7e-579e-6a97-a60c-dd46969f474e

### 

Query Parameters

**page** integer

Number of pages to return. Maximum: 20

**Example:** 1

**limit** integer

Minimum: 5 Maximum: 100

**Example:** 5
